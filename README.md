### Twitter clone - Twooter
---
### Services
    - nginx latest
    - php 7.4.X
    - mysql:8
    - redis:latest

### Build the containers 
`make build`    

### Start the containers 
`make start`    

___
*`http://localhost:PORT` should be available now* (configured in docker-compose.yml)

### Stop the containers 
`make stop`  

### Delete the containers 
`make clean`
